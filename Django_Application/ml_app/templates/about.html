About

This application analyzes uploaded videos to detect deepfakes using a CNN+LSTM pipeline.

Model

Feature extractor: ResNeXt50_32x4d → temporal head: LSTM → 2-class classifier (REAL / FAKE).

Demo mode

If model weights are missing or GPU inference isn’t available, the app can run in DEMO MODE (random prediction with a realistic confidence range) to validate the UI end-to-end.
